Goer Options

Configuration directives:

# comment
#!goer
If you want to load a script from a url, it should start with #!goer so we know we've hit the right url.
mark <name> <url>
Create a bookmark with the given name, that opens the given url. In order to use a bookmark, type either '<name> or `<name> to open the url in the current, or in a new tab.
mark <name> <location> <destination>
Create an association-mark between the <location> (a regular expression that matches a URL), and the <destination> (a replacement pattern, that can include $1 to $9 to extract parts from the location.
search <name> <url>
Create a search enging with the given name that replaces the %s in the URL by the search term. Type \<name> to start searching, and then either select, type or click on the word you wish to search for.
load <url>
Load a goer script from the url, it must start with #!goer.
log <message>
Log a message to the console
