Goer Options

Syntax

# comment
#!goer
If you want to load a script from a url, it should start with #!goer so we know we've hit the right url.
mark <name> <url>
Create a bookmark with the given name, that opens the given url. In order to use a bookmark, type either '<name> or `<name> to open the url in the current, or in a new tab.
mark <name> <context> <url>
Create a contextual bookmark that only takes effect when the current tab's URL matches the context. The destination can contain $1 through $9, allowing precise mappings between sets of urls.
In the case that multiple locations match the current one, the first defined is used.
search <name> <url>
Create a search enging with the given name that replaces the $1 in the URL by the search term. Type \<name> to start searching, and then either select, type or click on the word you wish to search for.
load <url>
Load a goer script from the url, it must start with #!goer.
